机房 Win10 专业版 虚拟机装 Win11 完整教程

一、先开启 Hyper‑V（Win10 专业版自带）

方法1：图形界面（机房最稳）

  1. Win + R 打开运行
  2. 输入 optionalfeatures 回车
  3. 找到 Hyper‑V，全部勾选：
    • Hyper‑V 管理工具
    • Hyper‑V 平台
  4. 确定 → 等待安装 → 重启电脑

方法2：PowerShell 命令（被锁界面时用）

以管理员身份打开 PowerShell，运行：

dism /online /enable-feature /all /featurename:Microsoft-Hyper-V

执行完成后重启。

二、Hyper‑V 入口在哪？

  • 开始菜单 → Windows 管理工具 → Hyper‑V 管理器
  • 或直接搜索：Hyper-V

三、把虚拟机存到 D 盘（防 C 盘爆炸）

  1. 打开 Hyper‑V 管理器
  2. 右侧点 Hyper‑V 设置
  3. 修改两个路径：
    • 虚拟硬盘 → D:\Hyper-V\VHD
    • 虚拟机 → D:\Hyper-V\VM
  4. 确定，以后自动放 D 盘

四、ISO 镜像下载（安全可用）

1. 官方 Win11（完整版 5–6GB）

https://www.microsoft.com/zh-cn/software-download/windows11

选择：Windows 11 磁盘映像 ISO → 下载

2. Tiny11 精简版（3GB 左右，机房首选）

GitHub：tiny11builder

网盘直下：

  • 夸克：点击下载
  • 百度：点击下载

五、新建 Win11 虚拟机（极简步骤）

  1. 新建 → 虚拟机
  2. 名称随便，位置选 D 盘
  3. 代数：第 2 代（Win11 必须）
  4. 内存：4096MB（4G） 以上
  5. 网络：Default Switch
  6. 硬盘：60GB 以上，存在 D 盘
  7. 安装选项：选择你下载的 ISO 文件
  8. 完成 → 启动安装

六、机房 360 注意事项

  • 360 提示“Hyper-V 正在运行” → 允许
  • 虚拟机内随便折腾，360 管不到
  • 机房重启会还原系统，但 D 盘虚拟机文件不会丢

开 Hyper‑V → 改路径到 D 盘 → 下 Tiny11 → 新建虚拟机 → 直接开玩
